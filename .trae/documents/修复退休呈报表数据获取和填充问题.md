# 问题分析

## 发现的问题

### 1. 数据库表结构不完整
`retirement_report_form` 表缺少退休呈报表需要的字段：
- `is_only_child` (是否独生子女)
- `join_party_date` (入党年月)
- `current_address` (现在住址)
- `work_experience` (工作简历)
- `retirement_reason` (退休原因)
- `family_members` (供养直系亲属)
- `retirement_address` (退休后居住地址)
- `pension_unit` (发给退休费的单位)

### 2. 后端API数据获取不完整
`/api/report-designer/teacher-full-data` 只查询了基本信息和教育经历，缺少其他数据。

### 3. 前端数据填充不完整
`RetirementReportForm.vue` 的 `open()` 函数只填充了部分字段。

## 修复方案

### 步骤1：修改数据库表
添加缺失的字段到 `retirement_report_form` 表。

### 步骤2：修改后端API
完善 `/api/report-designer/teacher-full-data` 接口：
1. 从 `teacher_basic_info` 获取基本信息
2. 从 `teacher_education_record` 获取教育经历
3. 从字典表转换代码为名称（民族、职务等）
4. 如果 `retirement_report_form` 表中有数据，优先使用

### 步骤3：修改前端
完善 `open()` 函数的数据填充逻辑。

## 实施步骤
1. 修改数据库表结构
2. 修改后端API
3. 修改前端
4. 测试数据是否正确显示
