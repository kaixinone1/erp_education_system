## 功能需求

### 1. 点击"填报"按钮时

* 打开数据管理界面 → 职工退休呈报表

* 自动预填充教师信息

* 如果重名，弹出选择框（显示姓名+身份证号）

### 2. 数据获取逻辑

* 优先使用档案出生日期

* 档案出生日期为空，则从身份证号提取

* 性别从身份证号提取（奇数男，偶数女）

### 3. 年龄判断退休

* 男：>=60岁

* 女：>=55岁（干部）/ 50岁（工人）

* 根据职务判断干部/工人

### 4. 实施步骤

#### 后端

1. 创建 API 获取教师详细信息（包含档案出生日期）
2. 创建 API 根据姓名搜索教师（处理重名）
3. 创建 API 生成退休呈报表数据

#### 前端

1. 修改 ChecklistDrawer 组件
2. 点击"填报"时：

   * 调用 API 获取教师信息

   * 如果唯一，直接预填充

   * 如果重名，弹出选择对话框

   * 打开数据管理器并传递预填充数据

### 5. 数据流

```
点击填报 → 获取教师信息 → 判断重名 → 
  ├─ 唯一 → 预填充 → 打开数据管理器
  └─ 重名 → 选择对话框 → 选中后预填充 → 打开数据管理器
```
