## 功能描述

在数据管理界面的任职状态列添加下拉菜单，实现快捷状态变更：

1. 在表格的"任职状态"列显示为下拉选择框
2. 用户点击下拉菜单选择新状态（如"退休"）
3. 如果选择的是"退休"，弹出确认对话框
4. 用户确认后，直接调用 API 更新状态
5. 更新成功后，自动触发清单推送

## 实施步骤

1. 修改 GenericDataView.vue 的表格列渲染逻辑
2. 为"任职状态"列添加特殊的下拉菜单渲染
3. 添加状态变更确认对话框
4. 调用 status-change API 更新状态并触发清单
5. 刷新表格数据

## 具体修改

在表格渲染时检测字段名，如果是"任职状态"，则显示下拉菜单而不是普通文本。
